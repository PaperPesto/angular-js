# - hash anchor

Fragment identifier

Il tag "a" ha l'attributo href che può indirizzare verso un url esterno oppure verso un elemento -interno- del DOM tramite l'hash #.

Lo script javascript riesce ad intercettare quando nell'url del browser c'è un cancelletto e stampa su console il valore appena dopo il cancelletto tramite window.location.hash

  • Provare a cliccare sugli ancor qui sotto e vedere sulla console l'output di hash-listener.js
  • Provare a cambiare a mano l'indirizzo del browser e vedere l'output su console
